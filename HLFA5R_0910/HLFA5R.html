Git parancsok

Git parancs Rövid leírás
git initÚj Git repó létrehozása a mappában.
git clone <url>Távoli repó klónozása helyi gépre.
git statusVáltozások és állapot megjelenítése.
git add <fájl>Fájl hozzáadása a staging területhez.
git commit -m "üzenet"Stagelt változások rögzítése.
git logCommit előzmények listázása.
git diffKülönbségek megjelenítése fájlok/commitok között.
git branchBranch-ek listázása/létrehozása (git branch név).
git switch <branch>Átváltás másik branch-re.
git merge <branch>Megadott branch beolvasztása az aktuálisba.
git pullTávoli változások letöltése és beolvasztása.
git fetchTávoli változások letöltése beolvasztás nélkül.
git pushHelyi commitok feltöltése a távoliba.
git stashFolyamatban lévő módosítások ideiglenes elmentése.
git stash popElrakott módosítások visszaállítása és törlése a stash-ből.
git restore <fájl>Fájl visszaállítása az utolsó commit/INDEX állapotra.
git revert <commit>Egy commit visszafordítása új commitban.
git reset --hard <commit>Visszaállás korábbi állapotra (FIGYELEM: adatvesztés lehetséges).
